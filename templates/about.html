About Blind Navigation AI

This application is a web-based AI system that detects objects in images and provides instructions or guidance to users. It is designed to assist users in understanding their surroundings by identifying objects accurately and generating meaningful instructions.

The system is built with Flask for the web interface, YOLOv8 for real-time object detection, and Python libraries for AI processing. Users can upload images, receive detailed detection results, and customize settings like language and volume for a personalized experience.
